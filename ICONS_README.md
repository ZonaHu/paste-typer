# Extension Icons

This extension requires icon files for proper display in Chrome. You'll need to create or obtain the following icon files:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Creating Icons

You can:

1. **Use an online icon generator** like https://favicon.io/ or https://www.canva.com/
2. **Design custom icons** using any image editing software
3. **Use the pen/typing emoji** 🖊️ as inspiration for the design
4. **Download free icons** from sites like https://icons8.com/ or https://www.flaticon.com/

## Temporary Solution

For testing purposes, you can:
1. Create simple colored squares in paint/photoshop
2. Save them as PNG files with the required dimensions
3. Place them in the extension directory

The extension will work without icons, but they improve the user experience.
